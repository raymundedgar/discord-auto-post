# Discord-GM-GN-bot

1. Clone
   ```bash
   git clone https://github.com/raymundedgar/discord-auto-post.git
   cd discord-auto-post
   ```
2. Env
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. Req
   ```bash
   pip install -r requirements.txt
   ```
4. Run
   ```bash
   python main.py
   ```
